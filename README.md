Blackwell Events – Event Management System

Blackwell Events is a web-based event management platform that allows organizations to create, manage, and publish events online. Users can browse events, view detailed information, and register for events. Administrators manage all event activities through a secure dashboard.

 Features

- Event listing and management
- Detailed event pages for each event
- Online event registration system
- Contact form with message storage
- Event flyer/image uploads
- Admin dashboard for managing events
- Ability to view registrations and user messages
- Responsive design for different screen sizes

Technologies Used

- PHP (Backend)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- InfinityFree Hosting (Deployment)

 Purpose

This project was built as a portfolio and academic project to demonstrate full-stack web development skills, including database integration, dynamic content handling, and admin system design.
